Project Overview

Going Solo is a mobile application designed to connect solo travelers, enabling them to find travel companions, share experiences, and discover new destinations together. The project focused on creating a safe, user-friendly platform that addresses the unique needs and concerns of solo travelers.

Project Goals

  • Create a safe and trustworthy platform for solo travelers
  • Design an intuitive matching system based on travel preferences
  • Build features that enhance travel planning and communication
  • Ensure accessibility and usability across different devices
